Expense Tracker

A simple web-based Expense Tracker application that allows users to manage their personal expenses. Users can add, edit, delete, and search expenses, with high-value expenses visually highlighted for quick tracking.

Features

Add Expenses: Enter details like title, category, amount, date, and payment method.

Edit & Delete: Update or remove existing expense records.

Search Functionality: Search expenses by title.

Highlight High-Value Expenses: Expenses above 1000 are highlighted in red.

Responsive Design: Built with Bootstrap 5 for mobile-friendly layouts.

Dynamic Table: Expenses are displayed in a sortable, interactive table.

Technologies Used

Frontend: HTML, CSS, JavaScript, Bootstrap

Backend API: RESTful endpoints (GET, POST, PUT, DELETE) assumed to be running on http://localhost:8080/expenses

Styling: Custom CSS for high-value expense highlighting

Installation

Clone the repository:

git clone https://github.com/yourusername/expense-tracker.git
cd expense-tracker

Ensure your backend API is running at http://localhost:8080/expenses.

Open index.html in your browser.

Usage

Add a New Expense: Fill the form and click Save.

Edit an Expense: Click Edit next to a row, update details, and click Save.

Delete an Expense: Click Delete next to a row.

Search by Title: Enter a title keyword and click Search.

Show All Expenses: Click Show All to reset the table.
